## Short Description
CampusConnect is a web app that connects students, teachers, and administration through one simple hub — events, study groups, marketplace, and announcements — improving campus life and communication.

## Tech Stack
- Frontend: React (Vite) + Tailwind CSS  
- Backend: Node.js + Express + Prisma  
- Database: PostgreSQL (Supabase)

## Features
✅ Events with RSVP  
✅ Study Groups  
✅ Lost & Found + Marketplace  
✅ Admin Dashboard
